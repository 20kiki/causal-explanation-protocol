# 安全策略

## 报告漏洞

如果你发现协议存在安全相关问题（如 prompt injection 漏洞、不当内容生成风险等），请通过 GitHub 私密渠道报告：

1. 前往 [Security Advisories](https://github.com/20kiki/causal-explanation-protocol/security/advisories/new)
2. 或发送邮件到 GitHub 个人资料中的公开邮箱

**请勿公开提交 Issue。**

## 适用范围

本安全策略适用于：
- SKILL.md 协议文件本身
- README 和文档中的示例
- Issue / PR 模板

## 本协议的定位

本协议是一个 LLM skill 文件——它作为系统指令注入到 AI 的上下文中。如果你发现协议中的某些措辞可能导致 LLM 产生意外行为，请按上述方式报告。
